# SPDX

> **noun.** A standard set of short, machine-readable identifiers for software and content licenses (for example `MIT` or `CC-BY-SA-4.0`), maintained by the Linux Foundation. Originally "Software Package Data Exchange," now used as a standalone name. The identifier list lives at <https://spdx.org/licenses/>.
